## Linux Calender

This is simple terminal base calender

# How to use:
```` console
$ git clone https://github.com/yahyakamran/cal.git
$ cd cal
$ make -s build 
````

Don't forget to put cal  into bin folder

# Usage:
```console
$ cal
```
#Requirements

clang is required if you want to use gcc you can edit MAKEFILE and use it.
